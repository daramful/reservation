#To run api (backend):
- cd api
- dotnet run

* Use Postman to check the status of endpoints (200, 400, 404, 500). At this initial development phase, there only should be 200. (return Ok())


#To run web (frontend):
- cd web
- npm start

Dev tools/frameworks needed:
- .net core 2.0
- react.js
- node.js
